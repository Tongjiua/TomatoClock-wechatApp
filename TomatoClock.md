# 微信小程序-番茄时钟

微信小程序版番茄时钟

## 概述：

微信小程序版番茄时钟（非官方出品）

* 数据来源：[番茄时钟] API
* 开发环境：Windows 10 64 bit
* 开发工具：微信web开发者工具 v0.11.112301
## 功能：

* 计时，规划工作休息时长
* 提前设置工作和休息的时间，自动提醒
* 可显示和清除时间记录
